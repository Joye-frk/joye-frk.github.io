# color-remixing

为有调色需求的毛娘打造的一个基于Web的水性漆调色工具，可以根据RGB值计算红、黄、蓝、黑、白五种基础色的混合比例。

## 功能特点

- 🎨 RGB颜色输入和实时预览
- 🔍 颜色名称搜索功能
- 📊 精确的调色比例计算
- 📱 响应式设计，支持手机访问
- 📝 详细的调色指导说明

## 使用方法

1. 访问 [调色助手页面](https://你的用户名.github.io/仓库名称)
2. 在"调色器"标签页输入RGB值，或
3. 在"颜色搜索"标签页搜索颜色名称
4. 查看调色比例和指导说明

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
